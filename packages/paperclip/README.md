# Paperclip Integration Package

Este paquete contiene los componentes UI de governance, budgets y organización para Lanzalo.

## Componentes

- **OrgChart** - Visualización de jerarquía de agentes
- **Budgets** - Tabla de budgets y progreso
- **Governance** - Control de agentes y configuración
- **Heartbeat** - Sistema de salud de agentes
- **Timeline** - Feed de eventos en tiempo real

## Integración con Lanzalo

Este paquete reusa los componentes existentes de Lanzalo y añade funcionalidades premium de governance y budgets como solicitó el CEO.
