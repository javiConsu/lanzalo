# POLSIA.COM — ESPECIFICACIÓN COMPLETA DE PRODUCTO

## RESUMEN EJECUTIVO
Polsia es una plataforma SaaS ($49/mo) que actúa como "empleado IA" autónomo. El usuario introduce una idea o su web existente, y la IA investiga al usuario, genera nombre de empresa, crea documentos estratégicos (Mission, Market Research), publica en Twitter, envía emails, despliega una landing page en un subdominio, y planifica tareas recurrentes — todo sin intervención humana. Modelo de monetización freemium con task credits. Facturan ~$2M.
